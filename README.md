# Sparkbox

PINS IN USE
-----------
SPI 1: 
PB2 - CS
PB5 - MOSI
PB4 - MISO
PB3 - SCK

SPI 2:
PB12 - CS
PB13 - SCK
PB14 - MISO
PB15 - MOSI

Push Buttons (for Sparkbox):
PD8 - 15

LEDs (on Discovery Board):
PE8 - 15

Push button (on Discovery Board):
PA0

UART1:
PC4 - TX
PC5 - RX

PWM (Audio):
PB6 - TIM4_CH1
PB7 - TIM4_CH2
PB8 - TIM4_CH3
PB9 - TIM4_CH4
